# 2019-stm32
本系统以STM32F103单片机为主控制器，以MaixCAM Pro为视觉识别单元，设计并制作了一套模拟电磁曲射炮（2019年电赛H题）。系统通过摄像头识别靶标像素坐标，经串口发送给STM32，计算偏差后控制两个张大头步进电机组成的二维云台精确瞄准，然后触发可控硅使高压电容对线圈瞬时放电，推动弹丸射出。采用双12V电池独立供电，有效隔离发射电路对控制电路的干扰。通过实测验得水平视场角64°、垂直视场角50°，建立了俯仰角—射程拟合函数，实现了手动输入距离打击（2~3米，偏差＜50cm）和自动搜寻打击功能。系统具有视觉闭环、激光测距辅助、模块化机械结构等特点，测试结果满足题目基本要求和发挥部分指标。
